🛡️ Phishing Awareness Analysis
A full-stack web application designed to detect and analyze suspicious URLs and help users understand potential phishing threats. The system evaluates input URLs using rule-based logic and provides a risk score along with an easy-to-understand safety report.
🚀 Features
🔍 URL-based phishing detection
⚠️ Risk scoring system (Low / Medium / High)
📊 Instant analysis results
🧠 Rule-based threat detection engine
📱 Simple and responsive user interface
🎓 Educational phishing awareness output

CSS
JavaScript
Backend:
